# Assets Directory

This directory should contain your app's image assets.

For a production app, you should add:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen image (1242x2436px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

## Quick Setup

You can use the default Expo assets for now. The app will work without custom assets.

To create simple placeholder assets:

1. Create a solid color 1024x1024 image for `icon.png`
2. Create a 1242x2436 image for `splash.png`
3. Copy `icon.png` to `adaptive-icon.png`
4. Create a 48x48 version for `favicon.png`

## Tools for Creating Assets

- [Figma](https://www.figma.com/) - Free design tool
- [Canva](https://www.canva.com/) - Easy image creation
- [MakeAppIcon](https://makeappicon.com/) - Generate all sizes
